# Members Only Blog

This is a blog app created with NodeJS and Express. Users can log in to write posts, but only select members can see the username of the author.

## How to set up

```bash
git clone git@github.com:mdesanker/members-blog.git
cd members-blog
npm install
npm run serverstart
```

## Built with

- NodeJS
- Express
- PassportJS
- bcryptJS
